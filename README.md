##### 目录说明：

- 家族树
  - bootstrap:bootstrap框架文件
  - css:网页的css样式文件
  - images:插入的图片
  - js:html文件中需要导入的一些js文件
  - type：html中需导入的css文件
  - 学术家族树：html文件  
- README:目录说明与使用说明文件

##### 使用说明：

- 点击github上的“Clone or download”打包下载。
- 使用时直接用Chrome打开**学术家族树.html**文件即可，输入格式按照作业要求即可。
- 在出现的文本框输入数据，点击生成家族树按钮，将会生成一棵以导师为根节点的树。
